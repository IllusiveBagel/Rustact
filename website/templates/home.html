Quick start

Spin up the demo and opt into live style reloads.

git clone https://github.com/IllusiveBagel/rustact
cd rustact
RUSTACT_WATCH_STYLES=1 cargo run

Prefer a fresh project? Use the bundled template:

cargo install cargo-generate
cargo generate \
  --git https://github.com/IllusiveBagel/rustact \
  --path templates/rustact-app \
  --name my-rustact-app

Why Rustact

Everything you need to craft feature-rich TUIs.

Component & hook system

Build apps with component("Name", render_fn) plus familiar hooks (use_state, use_reducer, use_effect, use_context, use_text_input).

  • Stateful views with memoized callbacks.
  • Provide/consume context for shared themes.
  • Text input bindings with validation helpers.

Async runtime & driver injection

Tokio-driven runtime handles terminal IO, ticks, events, and effect scheduling—with pluggable drivers for deterministic tests or headless mode.

  • Structured tracing for renders/events/shutdown.
  • Graceful task abort & Ctrl+C logging.
  • Test-friendly RuntimeDriver trait.

Rich widget catalog

Compose lists, tables, trees, forms, gauges, text inputs, tabs, modal overlays, toast stacks, and layered layouts without bespoke renderer code.

  • Mouse hitboxes for buttons/inputs.
  • CSS-inspired styling with styles/demo.css.
  • Live reload styles via RUSTACT_WATCH_STYLES.

Docs, template & automation

Comprehensive guides, tutorial, styling reference, API docs workflow, and a ready-to-run starter template.

  • GitHub Pages deploy with API docs bundled.
  • CHANGELOG.md + release checklist.
  • Starter template mirrors demo patterns.
